ALGORITHM :

  1. Start.
  2. Accept center (xc,yc) point and radius r of the circle
  3. Initialise:
    x = 0
    y = r
  4. Calculate initial decision parameter as:
    d = 3 – 2r
  5. while(x<=y)
    {
              Plotpoint(x,y); //use 8-way symmetry
             if d< 0
             then
                      x++;
                       y = y;
                       d = d + 4x + 6;
             else
                       x++;
                       y = y – 1;
                       d = d + 4(x – y) + 10
    }
  6. Stop.
